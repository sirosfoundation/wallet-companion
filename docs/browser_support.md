# Browser Support

Wallet Companion requires modern browser versions with Manifest V3 support.

## Minimum Versions

| Browser | Minimum Version | Release Date |
|---------|-----------------|--------------|
| Chrome  | 88+             | Jan 2021     |
| Firefox | 109+            | Jan 2023     |
| Safari  | 15.4+           | Mar 2022     |

## Why These Versions?

All browsers are pinned to versions that support **Manifest V3**, which provides:

- Unified extension APIs across browsers
- Improved security model
- Better performance with service workers

These minimum versions are 3+ years old as of 2026 and cover the vast majority of users.

Our recommendation would regardless be to keep your browser up to date with new versions and security patches.

## Checking Your Version

- **Chrome**: `chrome://settings/help`
- **Firefox**: `about:support`
- **Safari**: Safari menu / About Safari
